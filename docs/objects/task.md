# Task

## NAME

Task - "Runnable" subclass of kernel objects (threads, processes, and jobs)

## SYNOPSIS

[Threads](thread.md), [processes](process.md), and [jobs](job.md) objects
are all tasks. They share the ability to be suspended, resumed, and
killed.
